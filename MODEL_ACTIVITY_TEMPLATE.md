# Model Activity Template

Use this template for any new folder/model README so each model remains printable **and** learnable.

## Quick Start
- What to print first
- How to place/setup in under 2 minutes
- First observation target

## Student Mode
- Solo workflow in 10–20 minutes
- Notebook prompts: prediction, observation, explanation

## Mentor Mode
- Partner/coach workflow
- Suggested questioning sequence (without giving answers too early)

## Things to Predict Before Using
- 2–5 prediction prompts

## Hands-On Activities
- 3–8 activities
- Include a mix of manipulation, comparison, and measurement/counting tasks

## Explanation Prompts
- 2–5 prompts requiring verbal/written justification

## Common Mistakes / Misconceptions
- 2–4 misconception callouts specific to this model/topic

## Challenge Prompts
- 1–3 challenge prompts
- Include at least one comparison/constraint-style prompt when possible

## Physics 1 Connection
- Which AP Physics 1 units/skills this model supports

## F=ma Connection
- Which mechanics instincts this model builds (model choice, constraints, symmetry, limiting cases)

## Robotics Connection (if relevant)
- FTC/mechanism implications
- Likely real robot tradeoffs and failure modes

## Extension Ideas
- Parameter changes worth testing
- Suggested cross-links to other models/folders
