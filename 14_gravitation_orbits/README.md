# Gravitation and Orbit Geometry

Models: `orbit_geometry_template.scad`, `kepler_area_sweep_template.scad`

## What This Is
Templates for orbital shape geometry and area-sweep intuition.

## What It Teaches
Orbit geometry, focus-based intuition, and equal-area timing concepts.

## How To Print
Print templates flat with high-contrast marker grooves for manual tracing.

## Quick Start
Trace one ellipse, mark focus, and compare near-focus vs far-focus arc speed intuition.

## How to Learn With This
### Hands-on learning tasks
1. Mark periapsis/apoapsis regions and compare expected speed.
2. Use area-sweep sectors to reason about equal-time intervals.
3. Compare circular and elliptical limiting behavior.

## Things to Predict Before Using
- Where should orbital speed be greatest?
- What remains invariant in equal-time sweep idea?

## Challenge Prompts
- Explain why equal area in equal time implies varying speed along ellipse.

## Common Mistakes / Misconceptions
- Treating orbital speed as constant on all ellipse points.
- Confusing geometric center with force center.
