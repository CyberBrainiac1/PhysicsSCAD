# Fluids: Pressure and Buoyancy

Models: `pressure_with_depth_board.scad`, `buoyancy_comparison_block_set.scad`, `pascal_stack_press_plate.scad`

## What This Is
A depth-pressure board and buoyancy block set for conceptual fluid mechanics.

## What It Teaches
Pressure variation with depth, displaced volume intuition, and floating/sinking comparisons.

## Why This Matters
Fluids are often taught abstractly; these models make force distribution and displacement tangible.

## How To Print
Print depth board flat with clear tick marks; print block sets in multiple infills/masses for comparisons.

## Quick Start
Use the pressure-depth board first: mark two depths and predict which has greater pressure.

## How to Learn With This
### Student mode
- Run pressure activity + buoyancy activity in the same session for contrast.

### Mentor mode
- Ask “what variable changed: depth, volume displaced, or density?”

### Hands-on learning tasks
1. Pressure board: compare marker points at shallow vs deep levels.
2. Pressure board: count depth ticks and rank pressure qualitatively.
3. Buoyancy blocks: compare equal-volume blocks with different masses.
4. Buoyancy blocks: compare different-volume blocks with similar mass.
5. Pascal plate: discuss force transmission and area effects conceptually.

## Things to Predict Before Using
- At equal fluid, which point has greater pressure: deeper or shallower?
- Can a larger object sink while a smaller one floats?
- What changes buoyant force most directly in static fluid?

## Challenge Prompts
- Construct two blocks with equal buoyant force but different masses; predict outcomes.
- Explain a setup where pressure difference produces net upward support.

## Explanation Prompts
- Why does pressure depend on depth even in still fluid?
- Why is buoyancy tied to displaced fluid, not object weight alone?

## Common Mistakes / Misconceptions
- “Bigger always sinks less.”
- “Pressure is same everywhere in a container.”
- Confusing mass with density.
