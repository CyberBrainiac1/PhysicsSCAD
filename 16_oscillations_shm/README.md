# Oscillations and SHM Guides

Models: `pendulum_geometry_guide.scad`, `shm_displacement_phase_disk.scad`, `spring_mass_phase_slider.scad`

## What This Is
Geometry-first tools for pendulum angle motion and phase relationships in SHM.

## What It Teaches
Amplitude, phase, equilibrium restoring behavior, and period intuition.

## Why This Matters
SHM concepts become clearer when phase and geometry are manipulated directly.

## How To Print
Print phase disks and sliders with smooth marker surfaces for repeated annotation.

## Quick Start
On the phase disk, mark x=0 crossing points and predict velocity/acceleration directions there.

## How to Learn With This
### Student mode
- Alternate between pendulum and phase disk so geometry and graph views reinforce each other.

### Mentor mode
- Ask “where is restoring effect strongest/weakest?” before any formula use.

### Hands-on learning tasks
1. Pendulum guide: compare small vs large initial angles qualitatively.
2. Pendulum guide: identify turning points and equilibrium pass-through.
3. Phase disk: map displacement, velocity, acceleration phase offsets.
4. Slider: build one full cycle and mark quarter-cycle checkpoints.
5. Compare two amplitudes and discuss energy implications.

## Things to Predict Before Using
- Where is speed maximal in oscillation?
- Where is acceleration magnitude maximal?
- Does larger amplitude always mean same period for real pendulums?

## Challenge Prompts
- Construct a state with displacement zero but acceleration nonzero/zero? justify which is possible.
- Explain where small-angle approximation begins to degrade.

## Explanation Prompts
- Why are displacement and acceleration out of phase?
- Why does turning point have zero speed but nonzero restoring tendency?

## Common Mistakes / Misconceptions
- Treating SHM graphs as independent instead of phase-linked.
- Assuming pendulum is perfect SHM at any angle.
- Confusing equilibrium position with turning points.
