# Challenge Sets

Use these as short model-driven checks. Each set has beginner, intermediate, and harder prompts.

## Vectors Challenges
### Beginner
- Build two vectors with the same magnitude but different angle; predict which has larger x-component before measuring.
- Build a resultant from two arrows, then recreate the same resultant with a different pair.

### Intermediate
- Make three vectors that sum to zero without using opposite pairs.
- Keep resultant direction fixed; vary magnitude with component swaps.

### Harder (F=ma-style thinking)
- Construct two non-collinear vectors that cancel a third exactly; explain constraints geometrically.
- Find a setup where one component increases while magnitude stays constant.

## Kinematics Challenges
### Beginner
- Build x-t tiles for constant velocity and describe matching v-t.
- Build v-t with positive slope and identify acceleration sign.

### Intermediate
- Create two motions with same final position but different velocity history.
- Build motion with velocity crossing zero once; map to x-t turning behavior.

### Harder
- Build v-t where v=0 at one instant but a!=0, then explain physically.
- Match piecewise acceleration profile to a feasible motion narrative.

## Forces Challenges
### Beginner
- Set up FBD for level pull with friction.
- Set up FBD for incline rest case.

### Intermediate
- Show two situations where normal force differs from weight.
- Build a case where friction direction flips when tendency of motion flips.

### Harder
- Construct a static case where friction points “uphill” and defend why.
- Compare two force models with same net force but different internal balances.

## Friction Challenges
### Beginner
- Compare shallow vs steep ramp with same block.
- Compare smooth vs textured base setup.

### Intermediate
- Find angle window where static friction can hold but slight increase fails.
- Explain how normal force changes with angle and affects max static friction.

### Harder
- For two masses and same surface, predict which reaches threshold angle first and justify assumptions.
- Decide whether force-based or energy-based analysis is more efficient for a given incline prompt.

## Energy Challenges
### Beginner
- Place start markers at different heights; predict relative speed at bottom.
- Compare two paths with same start/end heights.

### Intermediate
- Identify where potential decreases fastest from track geometry.
- Explain why equal height drop does not guarantee equal final speed with losses.

### Harder
- Propose a test to separate geometric effects from dissipation effects.
- Build a scenario where energy framing is faster than force decomposition.

## Momentum Challenges
### Beginner
- Compare equal-speed tokens with different masses.
- Set up head-on equal-mass lane and discuss reversal expectations.

### Intermediate
- Hold momentum constant by trading mass and speed qualitatively.
- Compare 1D lane vs 2D board collision reasoning.

### Harder
- Create two collisions with same initial total momentum but different post-collision possibilities.
- Explain when momentum conservation helps more than energy conservation and why.

## Torque Challenges
### Beginner
- Balance equal masses at equal distances.
- Unbalance by moving one mass outward one slot.

### Intermediate
- Find two distinct balanced configurations with unequal masses.
- Keep torque constant while changing force and lever arm.

### Harder
- Determine minimal counterweight position to balance an offset load.
- Choose between torque summation and COM methods for a statics setup.

## Rotation Challenges
### Beginner
- Compare solid disk vs ring for spin-up intuition.
- Mark where mass is concentrated and predict resistance to angular acceleration.

### Intermediate
- Rank wheel families by expected spin-up speed under same applied torque.
- Relate shape choice to flywheel smoothing behavior.

### Harder
- Select mass distribution for max stored energy at fixed outer radius and mass budget (qualitative).
- Trade off cycle response vs speed stability for a robotics shooter concept.

## Center of Mass Challenges
### Beginner
- Find balance point for marked COM plate.
- Compare low vs high COM placement on stability board.

### Intermediate
- Use hidden COM variant and infer COM location from balancing attempts.
- Identify tipping edge for multiple payload positions.

### Harder
- Design payload placement that is stable in one direction but unstable in another.
- Explain COM projection criterion for tipping with clear geometry.

## Robotics Challenges
### Beginner
- Gear board: compare small-to-large vs large-to-small gear pair outcomes.
- Arm board: compare torque demand at short vs long extension.

### Intermediate
- Flywheel disks: pick geometry for consistency-first shooter behavior.
- Stability board: test COM shifts from intake extension concept.

### Harder
- Propose drivetrain/arm/flywheel parameter set for a specific match objective and defend tradeoffs.
- Given a tip-risk scenario, choose design changes in priority order and justify.
