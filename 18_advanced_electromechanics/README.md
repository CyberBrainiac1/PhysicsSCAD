# Advanced Electromechanics (Limited Scope)

Models: `dc_motor_torque_speed_concept_board.scad`, `magnetic_field_line_plate.scad`, `coil_motor_geometry_demonstrator.scad`

> Advanced enrichment only. Not core mechanics sequence.

## What This Is
Concept boards that connect a small amount of electromechanical intuition to robotics.

## Quick Start
Use the torque-speed concept board first; identify tradeoff direction before touching any motor data.

## How to Learn With This
### Hands-on learning tasks
1. Compare high-torque/low-speed and low-torque/high-speed regions.
2. Use field-line plate to discuss geometry vs force direction qualitatively.
3. Use coil geometry model to reason about why motor constants matter.

## Things to Predict Before Using
- What happens to available torque as speed rises?
- Which concepts transfer to choosing robot gearing?

## Challenge Prompts
- Explain one drivetrain decision that changes when torque-speed limits are respected.

## Common Mistakes / Misconceptions
- Treating motors as constant-torque devices at all speeds.
- Overextending simplified field visuals as exact quantitative tools.
