# Friction Ramp Kit

Model: `friction_ramp_kit.scad`

## What This Is
Incline wedges and block variants for friction/contact reasoning.

## What It Teaches
Static vs kinetic friction intuition, incline decomposition, and normal-force changes.

## Why This Matters
Friction problems become easier once you can picture force components and thresholds physically.

## How To Print
Print ramp family in multiple angles. Print one smooth and one textured block.

## Quick Start
Test a low-angle ramp and predict whether block stays at rest before placing it.

## How to Learn With This
### Student mode
- Increase angle one step at a time; record first-slip angle.

### Mentor mode
- Ask for force decomposition on ramp before each test.

### Hands-on learning tasks
1. Compare block behavior on shallow vs steep ramps.
2. Compare smooth-base vs textured-base block on same ramp.
3. Mark normal direction at multiple ramp angles.
4. Identify threshold angle where static friction fails.
5. Repeat with added mass and discuss expected/actual changes.

## Things to Predict Before Using
- How does normal force change as ramp angle increases?
- Does heavier always mean “slides easier” on same surface?
- What determines max static friction in each setup?

## Challenge Prompts
- Create two different setups that are both right at slip threshold.
- Explain a case where friction direction is opposite your first instinct.

## Explanation Prompts
- Why can static friction vary while kinetic friction is often modeled as fixed proportion?
- Why is normal force less than weight on an incline?

## Common Mistakes / Misconceptions
- Treating friction as constant in all static cases.
- Assuming weight component along ramp equals full weight.
- Forgetting friction can be zero if not needed.
