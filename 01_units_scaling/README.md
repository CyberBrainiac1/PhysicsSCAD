# Units and Scaling

## What This Is
Focused printable models for units and scaling.

## What It Teaches
Conceptual and geometric physics understanding via manipulatives.

## Why This Matters
Supports mechanics-first progression and links math representations to physical setups.

## How To Print
Use PLA at 0.2 mm layer height; avoid supports unless noted.

## How To Use
Print selected modules, run short guided activities, and compare parameter variants.

## Things To Notice
Small geometric changes often imply large conceptual changes.

## Try Changing These Parameters
Adjust dimensions, spacing, marker visibility, and labeling options.
