# Free Body Diagram Board

Model: `free_body_diagram_board.scad`

## What This Is
A force-placement board with center body region and directional arrow guides.

## What It Teaches
Force identification, direction discipline, and equilibrium/non-equilibrium reasoning.

## Why This Matters
Most mechanics errors start with bad force selection before any algebra happens.

## How To Print
Print board flat; print labeled force arrows as separate manipulatives.

## Quick Start
Place forces for a box on a level surface being pulled right at constant speed.

## How to Learn With This
### Student mode
- Draw FBD in notebook before placing arrows.

### Mentor mode
- Ask “what interaction causes this force?” for every arrow.

### Hands-on learning tasks
1. Build FBD for level pull with kinetic friction.
2. Build FBD for object at rest on incline.
3. Compare normal force between flat and incline cases.
4. Set a case with tension + weight + normal only.
5. Make one accelerating and one non-accelerating case with same listed forces but different magnitudes.

## Things to Predict Before Using
- Is normal force always equal to weight?
- In which cases can friction point in the +x direction?
- Which force disappears when no contact exists?

## Challenge Prompts
- Build a static incline case where friction points uphill.
- Build a case where two horizontal forces cancel but vertical forces do not.

## Explanation Prompts
- Why must every force correspond to an interaction?
- Why can friction oppose *tendency* of motion, not just actual motion?
- Why does net-force direction control acceleration direction?

## Common Mistakes / Misconceptions
- Adding “motion force.”
- Assuming normal equals weight in all situations.
- Drawing friction without identifying relative-slip tendency.
