# Contributing to PhysicsSCAD

## Principles
- Keep mechanics-first educational intent.
- Add one clear concept per model.
- Use reusable modules from `/common` whenever possible.
- Include learning guidance, not just geometry.

## Pull Request Checklist
- [ ] Model has clear educational purpose.
- [ ] Header + required comment sections included.
- [ ] Parameters are descriptive and documented.
- [ ] Printability checked for typical 220x220 mm printers.
- [ ] Folder README updated with usage notes and related models.
- [ ] Curriculum tags added/updated in `CURRICULUM_MAP.md`.

## Style and Structure
Follow `STYLE_GUIDE.md` for naming and formatting.

## Educational QA Questions
- What does the student physically compare or measure?
- How does this help AP1, F=ma, or FTC intuition?
- Is there a simpler geometry that teaches the same idea better?
